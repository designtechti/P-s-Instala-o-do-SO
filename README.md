## Automatização de instalação do Ubuntu

- Artigo com a explicação do funcionamento desse repositório no: [Criei um Sistema que se configura Sozinho | Marcelo Junior no LinkedIn](https://linkedin.com/pulse/criei-um-sistema-que-se-configura-sozinho-goulart-rodrigues-jr)

Clone o repositório:

	git clone https://github.com/designtechti/pos-instalacao-do-SO

Mude para o diretório em que o repositório foi baixado:

	cd pos-instalacao-do-SO/

Certique-se do arquivo ter permissões de execução.

	chmod +x ./script.sh

Para utilizar o Script, basta rodar o comando:

	sudo ./script.sh

Este repositório ainda apresenta uma lista de programas para se basear no download.
